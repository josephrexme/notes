# Getting Started with Quartz Notes

This is the first note in my new minimalist setup. 

Because of the configuration, no frontmatter is needed at the top of this markdown file. Just the title as a header, and you can jump straight into writing the content!

The build system will handle generating the layout and tying everything together. If I want to timestamp notes, I can just use a date prefix on the file names, like `2026-03-17-getting-started.md`.
