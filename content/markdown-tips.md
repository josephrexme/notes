# Markdown Tips

A quick reminder of some basic markdown syntax I might use:

- **Bold** text
- *Italic* text
- [Link to Google](https://google.com)

Code block:
```javascript
const example = "Hello, world!";
console.log(example);
```

As long as it's standard markdown (or GitHub Flavored / Obsidian), Quartz will render it beautifully on the live site.
